# Schoology-X

## <a>https://schoology-x.netlify.app/</a>

Chrome extension that layers productivity tools onto Schoology to keep upcoming work organized and visible.

### Features

-   Assignment checkboxes with saved state, strike-through, and a live progress bar (including support for revealed “more” items).
-   Add your own custom assignments with date/class tags; items sort by due date, slot into the upcoming list, and can be removed.
-   Quick helpers in the header: study timer shortcut and grades shortcut; header colors and logo can be customized and persist locally.
-   Course dashboard tweaks: rounded course cards and remembered course order.
-   Gradebook boosts: inline grade display, auto-open “Recently Completed,” and pop-up notifications when new scores arrive.
-   Keyboard shortcut to toggle the extension on/off: `Shift` + `Cmd/Ctrl` + `E`.
